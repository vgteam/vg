Detailed Description

Emit Alignments to a stream in SAM/BAM/CRAM format. Thread safe.

Create an HTSAlignmentEmitter writing to the given file (or "-") in the given HTS format ("SAM", "BAM", "CRAM"). Sample names and read groups for the header will be guessed from the first reads. HTSlib positions will be read from the alignments' refpos, and the alignments must be surjected.

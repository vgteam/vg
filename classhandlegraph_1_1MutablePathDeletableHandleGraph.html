Detailed Description

This is the interface for a graph which is deletable and which has paths which are also mutable.

handle_t handlegraph::MutablePathDeletableHandleGraph::change_sequence ( const handle_t &  handle,
const std::string &  sequence 
)
virtual

Change the sequence of handle's forward orientation to a new sequence. Returns a (possibly altered) handle to the node with the new sequence. The returned handle has the same orientation as the handle provided as an argument. May invalidate existing handles to the node. Updates paths to follow the new node if called through a class inheriting a MutablePathHandleGraph interface.
